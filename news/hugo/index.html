Hugo

by David Winterbottom on Thursday, 16 March 2017
by David Winterbottom
on Thursday, 16 March 2017

I’ve migrated this site to Hugo so I can host it on Github pages.

Hugo is a fast and well thought-out static site generator, written in Golang. It’s easy to learn - the trickiest part is understanding the difference between various ways pages are organised: “sections”, “types”, “taxonomies” etc.

The Vim plugin vim-markdown provides good support for authoring Hugo posts since it supports syntax-highlighting for:

  • Github-style “fenced” code blocks and
  • JSON or TOML front-matter.

The only frustration is having to use git submodules to keep the “public” folder of HTML files in a separate repo.
