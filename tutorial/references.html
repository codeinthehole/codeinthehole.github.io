6.0.2 Resources

  • CGLsim1D.m
    Matlab program that simulates the evolution of the one-dimensional CGL equation using pseudo-spectral code for the derivatives terms and exponential time-stepping for the time integration.

  • CGLsim2D.m
    Matlab program that simulates the evolution of the two-dimensional CGL equation using pseudo-spectral code for the derivatives terms and exponential time-stepping for the time integration. Here, a square container is assumed.
